🎵 Spotify Backend API

This project is a Spotify-like backend API built with Node.js, Express.js, and MongoDB.
It provides authentication, music upload, and album management features where artists can upload music and users can access it.

The backend uses JWT authentication, cookies, and cloud storage (ImageKit) to securely manage music files.

🚀 Features
🔐 Authentication

User Registration

User Login

User Logout

JWT Token Authentication

Password hashing using bcrypt

👤 Role System

User

Artist

Artists can upload music and create albums.

🎶 Music Management

Upload music files

Store music in cloud storage (ImageKit)

Fetch music list with artist details

💿 Album Management

Create albums

Add multiple songs to an album

Fetch albums and album details
